# AI Telegram Chatbot

An AI-powered Telegram chatbot built using Python, Aiogram, and OpenAI GPT API for real-time conversational interactions.

## Features
- Real-time AI-generated responses
- Telegram command support (`/start`, `/help`, `/clear`)
- Conversation context handling
- Asynchronous message processing
- Secure environment variable management using `.env`

## Technologies Used
- Python
- Aiogram
- OpenAI API
- python-dotenv
